# Enterprise Data Processing Platform (EDPP)

## Overview

Enterprise Data Processing Platform (EDPP) is a generic, metadata-driven backend application that ingests structured files, validates records, processes data, stores results in Oracle Database and exposes reporting APIs.

The project is intentionally domain independent and follows enterprise software engineering practices.

## Objectives

* Learn backend engineering through implementation.
* Build software using enterprise practices.
* Design a configurable platform.
* Evolve from Modular Monolith to Microservices.

## Tech Stack

* Java
* Spring Boot
* Oracle Database
* Spring Data JPA
* Maven
* Git
